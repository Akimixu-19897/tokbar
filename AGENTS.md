# tokbar 项目协作规则（必读）

## 沟通语言

- 与用户的所有沟通必须使用中文（简体），除非用户明确要求使用其它语言。

## 文档语言

- 在本仓库内新建或更新的文档（例如 `docs/` 下的设计/计划/说明等）必须使用中文（简体）。

## 代码注释

- 新增或修改的代码必须包含必要注释，并且注释必须使用中文（简体）。
- 注释要求以“解释意图/约束/边界条件”为主，避免无意义的逐行复述。

## Skills（本地技能）

- 全局 skills 已安装到当前工作区：`.codex/skills/`。
- 当任务与某个 skill 描述匹配或用户点名 skill 时，必须优先遵循对应 `SKILL.md` 的流程与约束。
